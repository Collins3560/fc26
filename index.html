Installation Instructions (PS4 & PS5)

  1. Download the 3 updated squad files from this page (updated regularly)
  2. In EA SPORTS FC 26 go to Customise → Profile → Save Squads
    Create a new save with an English name.
    If you get a save error → restart console and try again.
  3. Download & install Apollo Save Tool:
    https://pkg-zone.com/details/APOL00004
  4. Insert USB → Run Apollo Save Tool → HDD Saves → Find your EA SPORTS FC 26 save → Copy Save to USB
  5. On PC: Open the USB save folder → Replace the 3 files with the ones you downloaded
    ⚠️ NEVER touch the sce_sys folder!
  6. Insert USB back → Apollo Save Tool → USB Saves → Select your save → Copy Save to HDD
  7. Launch game → Customise → Profile → Load Squads → Choose your saved squad → Done!
